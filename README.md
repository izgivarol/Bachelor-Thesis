# Bachelor Thesis – Quantitative Synaptic Analysis

This repository contains all analysis files used in the quantitative evaluation of synaptophysin puncta associated with BDA-labeled corticospinal tract fibers.

## Repository Structure

- **01_CellProfiler_Pipeline/**
  - Custom CellProfiler pipeline used for object detection and spatial analysis

- **02_Image_Analysis_Output/**
  - Image-level summary outputs generated by CellProfiler

- **03_Data_Processing_R/**
  - R scripts used for data cleaning, aggregation, and calculation of quantitative metrics

- **04_Final_Dataset/**
  - Final animal-level dataset used for statistical analysis (GraphPad Prism)

## Notes

- Quantification was performed on the intermediate gray matter (ventral spinal cord)
- Data were aggregated at the animal level prior to statistical analysis
